# 峰云共享系统 - 更新日志

---

## [2.0.0] - 2025-04-12
### 重大更新
- 完全重构系统架构，从文件存储迁移到数据库存储
- 引入完整的用户生命周期管理

### 新增功能
#### 用户系统
- ✅ 用户注册功能（支持邮箱验证）
- ✅ 数据库集成（MySQL存储用户凭证）

#### 前端改进
- 🎨 响应式设计（适配移动设备）
- ✨ 页面加载动画（基于animate.css）
- 📅 动态版权年份显示

#### 安全增强
- 🔒 密码加密存储（bcrypt算法）
- ⚠️ 账户锁定机制（3次失败尝试后锁定）

### 技术变更
- 🔧 重构认证流程核心逻辑
- ⬆️ 升级Flask至3.1.0版本
- ➕ 新增PyMySQL数据库驱动依赖
